# Demo Package

`demo-source.txt` を1,000 bytes単位で分割した確認用パッケージです。

## Webで復元する

1. GitHub Pagesで公開したSplitSendを開きます。
2. 「復元」タブを選択します。
3. `SplitSend_demo-source_D3E02026` 内の `.partXXX` と `.splitsend.json` をまとめて選択します。
4. 復元先を選びます。
5. 復元された `demo-source.txt` と、このフォルダー直下の `demo-source.txt` が一致することを確認します。

マニフェスト記載のSHA-256が一致すれば、バイト単位で完全に同一です。
